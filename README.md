# Muhammad Reky — Portfolio 2026
> Software Engineer & Digital Designer.

A high-performance personal portfolio website designed to showcase the intersection of design-centric frontend development and machine learning integration. Built with a focus on modularity, motion physics, and minimalist aesthetics.

---

## Technical Foundation
The architecture is optimized for smooth interactions and rapid deployment using modern web standards.

* **Core Framework:** React 18 (Vite)
* **Motion Engine:** GSAP (GreenSock Animation Platform)
* **Style Architecture:** Tailwind CSS (Utility-first)
* **State Management:** React Hooks & Context API
* **Graphics:** HTML5 Canvas API

---

## Key System Integrations

### 01. Cyber Command Palette
An OS-level navigation interface accessible via `Ctrl + K`. Designed for workflow efficiency, allowing users to instantly access Resume (PDF), copy contact credentials, and perform section jumping without manual scrolling.

### 02. Interactive Vandalism Mode
A custom HTML5 Canvas implementation that enables real-time user-interaction directly on the DOM. This feature demonstrates proficiency in browser-based graphics rendering and state persistence during scroll events.

### 03. Liquid Theme Engine
A non-linear theme switching system using GSAP-driven ripples and CSS Invert logic. It ensures visual consistency and accessibility across Dark/Light modes while maintaining high-frame-rate transitions.
